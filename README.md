# 🤥 TINA

> Developer excuses generator. Because every bug deserves an explanation.

TINA will generate funny programmer excuses directly from your terminal.

## Installation

Run:

```bash
npx @roxytina/devlies
```

or install:

```bash
npm install -g @roxytina/devlies
```

---

## Usage

Random excuse:

```bash
devlies
```

Example:

```
🤥 TINA

The database needed emotional support.

Category:
backend
```

---

## Commands

### Random

```bash
devlies random
```

### Roast yourself

```bash
devlies roast
```

### AI excuses

```bash
devlies ai
```

### Developer horoscope

```bash
devlies horoscope
```

### Daily excuse

```bash
devlies daily
```
